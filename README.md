# Projet ARYHA

Ceci est la repository du projet qui sera présenter à l'épreuve du Titre de Developpeur Web.

### Stack utilisée
Utilisation du Framework Symfony.
Les bibliothèque Bootstrap / Jquery.

#### Pour récupérer le projet
Télécharger le fichier zip de mon prejet.

Ouvrir le terminal dans le dossier ARYHA puis taper :
  $ composer install
  
Ouvrir PhpMyAdmin et créer un projet du nom de ARYHA.
Importer le fichier .sql mis à disposition dans le projet.
